תאור

ברוכים הבאים לעולם הקוד! בשנה שעברה מיליוני לומדים מהקהילה שלנו התחילו עם HTML. למה? HTML הוא השלד של כל דפי האינטרנט. לרוב היא השפה הראשונה שלמדו מפתחים, משווקים ומעצבים והיא מהווה בסיס לעבודה בפיתוח חזיתי. אם זו הפעם הראשונה שאתה נוגע בקוד, אנו נרגשים לקראת מה שאתה עומד ליצור. אז מהו בעצם HTML? HTML מספק מבנה לתוכן המופיע באתר, כגון תמונות, טקסט או סרטוני וידאו. לחץ לחיצה ימנית על כל דף באינטרנט, בחר "בדוק" ותראה HTML בחלונית במסך. HTML מייצג שפת סימון HyperText: שפת סימון היא שפת מחשב המגדירה את המבנה והצגת הטקסט הגולמי. ב- HTML המחשב יכול לפרש טקסט גלם עטוף באלמנטים של HTML. HyperText הוא טקסט המוצג במחשב או בהתקן המספק גישה לטקסט אחר באמצעות קישורים, המכונה גם היפר -קישורים. סביר להניח שלחצת על כמה היפר -קישורים בדרך לקורס Codecademy הזה. לימוד HTML הוא השלב הראשון ביצירת אתרים, אך אפילו מעט ידע יכול לעזור לך להזרים קטעי קוד לניוזלטר, בלוג או תבניות אתר. ככל שתמשיך ללמוד, תוכל לשלב HTML עם CSS ו- JavaScript כדי ליצור אתרים משכנעים ודינאמיים. אבל לעת עתה, נתמקד כיצד להוסיף ולשנות תוכן בסיסי בדף, כמו טקסט, תמונות וסרטונים. אל תדאג אם האתרים נראים מכוערים - אנחנו רק מתחילים.
